
Plantilla DOCX para presupuestos Muvex
--------------------------------------
1. Colocar aquí el archivo 'presupuesto.docx' original.
2. Usar los placeholders:
   {{nro}}, {{fecha}}, {{cliente}}, {{empresa}}, {{ruc}}, {{telefono}},
   {{desde_cabecera}}, {{hasta_cabecera}}, {{srv_desde}}, {{srv_hasta}},
   {{std_num}}, {{std_letras}}, {{int_num}}, {{int_letras}},
   {{est_m3}}, {{est_tiempo}}, {{fecha_sugerida}}.
3. Este archivo se reemplaza en cada generación y se convierte a PDF.
